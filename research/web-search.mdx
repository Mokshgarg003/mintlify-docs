## What Search Does

Search retrieves current information from the web to inform design decisions. It finds data, benchmarks, trends, and context that make outputs more relevant and accurate. Search provides:

- Current industry data
- Competitor information
- Best practice research
- Trend analysis
- Factual verification

Figr engages web search when you:

- Understanding the market, what needs to designed
- Existing solutions for problems
- Ask about current trends or data
- Need industry benchmarks
- Want competitive information
- Request recent examples
- Ask questions requiring current facts

Select & Edit changes one part of a screen, a section, component, layout area, or piece of copy, and leaves the rest exactly as it was. Most AI tools regenerate the whole screen for one small change, so fixing the button shifts the layout and improving the copy breaks the spacing. To use it, select the part you want to change on the canvas, then describe the edit. Figr applies it to that selection only. Reach for full regeneration when the direction itself is wrong, and Select & Edit when the direction is right but the details need work.

Good refinement prompts:

- "Make this warning clearer."
- "Use the side panel pattern from this reference."
- "Add the empty state for users with no integrations."
- "Make only this table section more compact."
- "Change the hierarchy in this card without changing the rest of the page."

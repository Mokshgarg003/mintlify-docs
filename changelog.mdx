## November 2024

### Version 2.7.2 - Nov 03, 2025

**Features**

- **Self-service design intelligence setup.** Configure design system guidelines and product context generation directly from your workspace. No need to wait for manual configuration.

**Improvements**

- **Enhanced design workflow interactions.** Visual nudge when designs appear, tooltips on hover for context, and relocated select-and-edit button for easier access.
- **Attachment support in select-and-edit mode.** Add reference images and files while editing designs.
- **Narrower chat sidebar minimum width.** More canvas space for design work without compromising chat readability.

**Fixes**

- Resolved state reversion issue when using stop-and-edit functionality.
- Fixed mobile overlay display inconsistencies.

### Version 2.6.0 - Oct 15, 2025

- **Retry button for failed actions.** Quickly retry operations without starting over.

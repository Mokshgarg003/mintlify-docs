## April 2026

### Version 3.2.0 - April 7, 2026

<Card title="Major Release" icon="atom-simple" iconType="regular">
  Agent 2.0 is live in beta. Figma plugin for live sync.
</Card>

**Features**

- **Agent 2.0 is live in beta.** Figr now builds a complete file system for your project, goes significantly deeper into multi-screen design, and handles complex prototypes in a single session. Editing is faster, more precise, and the agent can run longer on ambitious builds.
- **Figma Plugin for live sync.** No more pasting Figma links one by one. Open the Figr plugin inside any Figma file and Figr instantly accesses the complete file, including tokens, components, and any frame you select, with real-time sync.
- **Refreshed canvas experience.** Comments, sticky notes, artifact rearranging, deleting, and full canvas management are now built in. Your canvas is a proper workspace.

**Improvements**

- **Instant retry.** Retry button now starts streaming immediately. Updates are offloaded to the background.
- **Optimized canvas loading for deep agents.** Files load incrementally, canvas state can be modified mid-stream, and layout algorithm is improved.

**Fixes**

- Canvas state saving bug resolved.

## March 2026

### Version 3.1.0 - Mar 26, 2026

**Features**

- **Deep Agent available for all Pro users.** The deep agent, previously in limited access, is now live for every Pro subscriber. Longer-running, more complex design projects with multi-screen output.
- **Context Pods (formerly Product Ally).** Renamed for clarity. Same powerful context system, now called Context Pods across the product. Includes much better processing and fetches specific parts of the context to save token cost.

**Improvements**

- **Significantly faster app performance.** First load, project pages, chat pages, and Context Pods all load noticeably faster. Projects are paginated. Chat canvases load incrementally as you scroll instead of all at once.
- **All lists virtualized.** Long project lists, chat histories, and artifact collections no longer cause lag regardless of size.
- **Better error messages.** Errors from the backend now surface with specific, actionable messages instead of generic failures.

**Fixes**

- Figma copy behavior fixed.
- XML code no longer appears in outputs.
- Permanent loading states now have retry handling.

### Version 3.0.6 - Mar 3, 2026

**Features**

- **Edit subagent for design changes.** A dedicated agent now handles design edits with better context isolation. Edits are more precise and don't bleed into unrelated parts of your design.
- **Video analysis in execution agent.** The execution agent can now process video attachments directly during design work. Share a screen recording mid-flow and Figr incorporates it without switching modes.

**Fixes**

- Regenerate button now appears for errored states in spaces after 10 minutes.
- Credits modal copywriting updated for clarity.
- Various stability optimizations throughout the app.

## February 2026

### Version 3.0.4 - Feb 23, 2026

**Features**

- **Product context images in designs.** Figr's design agent can now embed your uploaded images, logos, screenshots, and brand assets, directly into generated prototypes. No more placeholder swaps after the fact.

**Improvements**

- **Space-based filters in All Chats.** Filter your chat history by space. Easier to find past conversations when you're working across multiple products.
- **Product Ally navigation via canvas.** The active Product Ally now displays beside the project name in canvas, so you always know which product context Figr is referencing.
- **Smarter context management.** Context cleanup is now phase-based instead of aggressive. Prevents premature cache invalidation, which means fewer repeated computations and more stable credit usage mid-session.
- **Faster design edits.** The execution agent now uses targeted reads (grep → offset → edit) instead of reading full files. Roughly 5x less context consumed per edit.
- **Better video analysis.** Improved model and analysis for screen recording interpretation. Figr extracts more accurate context from video walkthroughs.
- **More reliable image reading.** Improved handling for fetching reference images from URLs with graceful fallback when sources are inaccessible.
- **Credit deduction improvement.** More accurate billing, fewer surprise spikes.
- **Workspace rename permissions.** Only workspace owners and admins can rename workspaces now.
- **Persistent filters and org selection.** Project filters you apply now persist across sessions. Your last opened org stays selected even after logout.

**Fixes**

- File handling overhaul:
  - File names are sanitized on upload. Special characters replaced with `_` so uploads don't silently break.
  - Original file type is detected even if someone manually renames the extension. An `image.png` renamed to `image.webp` is still treated as PNG.
  - Duplicate file names no longer overwrite each other.
  - Only supported image formats accepted: JPG, JPEG, PNG, WebP, and GIF.
- Markdown node wasn't switching versions on first attempt. Fixed.
- Intercom integration to canvas only.
- Human assistance intermittent failures resolved.

# CSV & Excel Import

Real data creates real designs. Import your actual spreadsheets, user data, and analytics to help Figr understand your content structure and create designs with realistic, representative data.

<tip>
  **Data-driven design:** Designs created with real data look more convincing, reveal layout challenges early, and help stakeholders envision the final product accurately.
</tip>

## Why Real Data Matters

<CardGroup cols={2}>
  <Card title="Before: Lorem Ipsum Design" icon="file-text">

    - Unrealistic layout assumptions

## Best Practices for Data Import

<CardGroup cols={2}>
  <Card title="Data Quality" icon="shield-check">
    **Prepare quality data:**

    ‚úÖ **Clean data** before import (remove duplicates, fix errors)
    ‚úÖ **Representative sample** (include edge cases and variations)
    ‚úÖ **Current data** (recent enough to be relevant)
    ‚úÖ **Complete records** (minimal missing values)
    ‚úÖ **Diverse examples** (different user types, scenarios)
  </Card>
  <Card title="Privacy First" icon="lock">
    **Protect sensitive information:**

    ‚úÖ **Remove unnecessary PII** before upload
    ‚úÖ **Use test/demo data** when possible
    ‚úÖ **Enable anonymization** for real data
    ‚úÖ **Check team permissions** before sharing
    ‚úÖ **Review data retention** settings regularly
